Álgebra

Equações, inequações, funções e polinômios. Explora as relações entre variáveis e constantes.

Exemplo de Fórmula:

Equação quadrática: ax² + bx + c = 0

Fórmula de Bhaskara: x = (-b ± √(b² - 4ac)) / 2a

Geometria

Estudo das formas, tamanhos e propriedades das figuras e espaços.

Exemplo de Fórmula:

Área de um círculo: A = πr²

Teorema de Pitágoras: a² + b² = c²

Cálculo

Limites, derivadas e integrais. Ferramenta essencial para modelar mudanças.

Exemplo de Fórmula:

Derivada de uma função: f'(x) = lim(Δx → 0) (f(x + Δx) - f(x)) / Δx

Integral de uma função: ∫ f(x) dx = F(x) + C
