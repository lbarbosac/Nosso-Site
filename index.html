Álgebra

Equações, inequações, funções e polinômios. Explora as relações entre variáveis e constantes.

Exemplo de Fórmula:

Equação quadrática: ax² + bx + c = 0

Fórmula de Bhaskara: x = (-b ± √(b² - 4ac)) / 2a

Geometria

Estudo das formas, tamanhos e propriedades das figuras e espaços.

Exemplo de Fórmula:

Área de um círculo: A = πr²

Teorema de Pitágoras: a² + b² = c²

Trigonometria

Análise dos ângulos e suas relações com os comprimentos dos lados de triângulos.

Exemplo de Fórmula:

Sen, Cos, Tan de um ângulo: sen(θ) = oposto/hipotenusa, cos(θ) = adjacente/hipotenusa, tan(θ) = sen(θ)/cos(θ)

Identidade trigonométrica: sen²(θ) + cos²(θ) = 1

Cálculo

Limites, derivadas e integrais. Ferramenta essencial para modelar mudanças.

Exemplo de Fórmula:

Derivada de uma função: f'(x) = lim(Δx → 0) (f(x + Δx) - f(x)) / Δx

Integral de uma função: ∫ f(x) dx = F(x) + C

Estatística

Coleta, análise, interpretação e apresentação de dados.

Exemplo de Fórmula:

Média aritmética: μ = (Σxi) / n

Desvio padrão: σ = √(Σ(xi - μ)² / n)
